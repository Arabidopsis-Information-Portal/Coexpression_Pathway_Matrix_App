D3 Matrix Demo

This is a demonstration application developed by Maria Kim for the scientific investigation of co-expression data. Each tab displays a matrix of Arabidopsis thaliana gene co-expression values from ATTED-II that are within a selected Kegg Pathway class, subclass or pathway. This application uses the D3 JavaScript library to build the matrices. Each matrix can be manipulated with three parameters; display (KEGG class, subclass, or pathway), a correlation coefficient filter, and an order. The data is not live or dynamic as it is stored in local files in the JSON format.
